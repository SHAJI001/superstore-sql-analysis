# Superstore SQL Analysis

## Overview
This project uses PostgreSQL to analyze a cleaned Superstore sales dataset with 9,994 transaction records. The analysis focuses on revenue, profit, discount behavior, regional performance, customer segments, shipping performance, and product profitability.

## Tools Used
SQL
PostgreSQL
Data Validation
Exploratory Data Analysis
Window Functions
Indexing and Query Optimization

## Project Objectives
Analyze sales and profit trends across regions, categories, and customer segments

Evaluate how discounting impacts profitability

Identify high sales but low profit product groups

Use advanced SQL techniques such as grouped aggregates, subqueries, CTE logic, and window functions to answer business questions

## Key Findings
The West region produced the highest sales and profit

Technology was the most profitable category

Very high discounts were associated with negative average profit

Some subcategories generated high sales but weak or negative profit

Consumer led in revenue while Home Office had the strongest profit margin
